Ako upraviť zobrazovanie "search box" vo vlastnej téme vzhľadu

18 máj, 2011 - 07:22 poslal salsastevie

Zdravím,

robím si vlastnú tému vzhľadu a narazil som na takýto problém.

searchbox, ktorý sa dá vložiť do témy má nejaké svoje predvolené zobrazenie, kde je text "Hladať na tejto stránke:", inputbox a tlačidlo hľadať.

Ja by som potreboval aby tam text "Hľadať na tejto stránke:" nebol a aby miesto tlačidla bol obrázok.

Podarilo sa mi zrušiť text "Hľadať na tejto stránke:" v CSS tým, že som ho dal display: none, no text "hľadať" zameniť za obrázok neviem.

Nedávno som ešte čítal, že google penalizuje stránky ak je tam "skrytý" nejaký text cez CSS.

Prihladám obrázok toho, čo sa mi podarilo spraviť cez CSS (obrázok dole) a ako by som to chcel zobraziť (obrázok hore).

Je možné, že už je tento problém už niekde popísaný, no neviem, čo mám hľadať aby som ho našiel.

Ďakujem za každú radu prípadne nejaký link.

Takže… Vôbec netreba search-.....tpl.php

Treba len vytvoriť v adresári témy súbor template.php

a doňho dať toto:

<?php

/**
 * Modify the theme search box.
 */
function názovtémy_preprocess_search_theme_form(&$vars, $hook) {
  // Remove the search box title.
  unset($vars['form']['search_theme_form']['#title']);

  // Replace the submit button with an image.
  $theme_path = drupal_get_path('theme', 'názovtémy');
  $vars['form']['submit'] = array('#type' => 'image_button', '#value' => t('Search'),
                             '#src'  => $theme_path . 'cesta k obrázku, v adresári témy, ktorých chceme mať na submit tlačitle');

  // Rebuild the rendered version (search form only, rest remains unchanged)
  unset($vars['form']['search_theme_form']['#printed']);
  $vars['search']['search_theme_form'] = drupal_render($vars['form']['search_theme_form']);

  // Rebuild the rendered version (submit button, rest remains unchanged)
  unset($vars['form']['submit']['#printed']);
  $vars['search']['submit'] = drupal_render($vars['form']['submit']);

  // Collect all form elements to make it easier to print the whole form.
  $vars['search_form'] = implode($vars['search']);
}

Miesto „názovtémy“ dať názov svojej témy a miesto „cesta k obrázku, v adresári témy, ktorých chceme mať na submit tlačitle“ dať to čo sa tam píše… :-)

Samozrejme treba si obnoviť tému aby sa to prejavilo… Ja to robím tak, že sa prepnem na inú a potom zase na tú svoju. :-) Ak existuje ešte nejaký iný spôsob, rád sa nechám poučiť…

:-) prišiel som na to tak, že som odkukal ako je to spravené v téme, ktorú používa aj www.drupal.sk :-)

Ďakujem.

K tomu textu vo vyhľadávacom poli, ktorý má zmiznúť po kliknutí a objaviť sa po strate focusu - Ja mám v template.php vo funkcii nazovtemy_preprocess_search_theme_form (okrem iného) aj tento riadok:

$vars['form']['search_theme_form']['#value'] = t('napíšte hľadaný text a stlačte enter');

a cez funkciu drupal_add_js pridaný javascriptový (jQuery) súbor, v ktorom je (okrem iného) aj:


$('#search input').focus(function() {
$(this).val('');
});
$('#search input').blur(function() {
$(this).val('napíšte hľadaný text a stlačte enter');
});

Easy. :-)
