Ahojte. Tak som sa po dlhom čase povenoval nejakým veciam a preskúmal som (porovnávaním databázy z inej inštalácie) a našiel kde bol problém. Problém je eliminovaný…

Pokiaľ by to niekoho zaujímalo, bolo to tým že v PostgreSQL zostala nadefinovaná funkcia:

CREATE OR REPLACE FUNCTION if(boolean, anyelement, anyelement)
  RETURNS anyelement AS
    'SELECT CASE WHEN $1 THEN $2 ELSE $3 END;'
  LANGUAGE 'sql' VOLATILE;

a mala byť:

CREATE OR REPLACE FUNCTION if(boolean, text, text)
  RETURNS text AS
    'SELECT CASE WHEN $1 THEN $2 ELSE $3 END;'
  LANGUAGE 'sql' VOLATILE;
