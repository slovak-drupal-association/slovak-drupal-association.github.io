TinyMCE

12 apríl, 2010 - 14:58 poslal ruso

Ahoj,

nepoznám Drupal veľmi dobre, ale snažím sa na ňom rozbehnúť komunitné stránky herného klanu. Systém mám nainštalovaný, tému aj všetky ostatné záležitosti mám hotové. Problém však nastal keď som sa pokúsil inštalovať WYSIWYG editor.

nainštaloval som nasledovne moduly:

-IMCE 6.x-1.3
-IMCE Wysiwyg API bridge 6.x-1.1

a aplikáciu TinyMCE , ktorú som inštaloval do adresára /sites/all/modules/includes/tiny_mce

Problém je že modul IMCE Wysiwyg API bridge oznamuje chybu :

"Makes IMCE available as plugin for client-side editors integrated via Wysiwyg API. Závisí na: IMCE (povolený), Wysiwyg (chýbajúci)"

Z toho usudzujem, že modul nevie nájsť súbory TinyMCE.

Skúšal som ich nakopírovať už aj do adrresára /www/drupal/sites/all/libraries/includes, takisto aj do spomínaných adresárov bez toho includes.

Viete mi niekto poradiť ako teda WYSIWYG editor na Drupal správne nainštalovať?

Ďakujem za pomoc

TinyMCE sa instaluje do "sites/all/modules/tinytinymce"

Potrebujes nato modul a balicek. Samozrejme, ze ked uz, tak aj sk alebo cz language pack.

V sites/all/modules nemas mat adresar includes a libraries. Respektive netusim, preco alebo pre ktory modul to tam mas. Este som sa stym nestretol.

Na pouzivanie IMCE Wysiwyg API bridge potrebujes wysiwyg a IMCE modul. Neviem ci to funguje aj s TinyMCE a nechce sa mi to ani skumat.

Treba citat README.txt a INSTALL.txt subory. Je tam vacsinou popis instalacie.

PS: o klanstranke si mozme pokecat cez email. Mam uz jednu rozbehnutu, ale potrebujem este zopar vylepseni a "ficurin" z inych CMSiek.

Ked pouzivas modul Wysiwyg, potrebujes iba ten balicek. Modul tinyMCE neoporucam, konfiguruje sa ovela tazsie ako modul wysiwyg, ktory je novsi a podporuje celu plejadu editorov.

folder, ktory kolega uvadza, sa pouzival pri module tinyMCE, ale pri module wysiwyg uz nie.

Pre wysiwyg treba tinyMCE nainstalovat do foldra sites/all/libra­ries. Potom je v nastaveni Wysiwyg uz mozne zvolit TinyMCE ako editor.

Inak,vzdy pomoze precitat si pribaleny README, kde su najaktualnejsie informacie, a kde je ta cesta do libraries uvedena. Instaloval som to niekolko krat, zatial zadne problemy.

K vysiwyg modulu odporucam este pribalit dalsi modul: better formats, ktory pomoze pri nastaveni input formatov pre rozne pouzivateske roly.
