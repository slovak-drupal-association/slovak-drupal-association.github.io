Uprava temy CSS views(Theme information)

31 január, 2011 - 17:33 poslal joeko

Zdravim

Chcem zmenit vzhľad poli v Bloku. Blok som vytvoril cez views. Tu zmenu poli vraj mozem spravit cez views kliknutim na Theme Information. Vo views som klikol na „row style output“ a vytvoril subor „views-view-fields–poc–block-1.tpl.php“ podla video navodu (http://www.maybeor.com/…-rows-tables)a skopiroval do adresara k teme. Avsak subor php som trochu pomenil pridal tam nejake CSS kod ale nefunguje mi to. Mozte prosim skontrolovat subor ci som to dobre vytvoril, pretoze k ziadnej zmene vzhladu poli nedoslo po skopirovani suboru do adresara temy.

Mozte tiez poradit co v tom subore treba spravit, aby som zmenil font velkost farbu v Bloku. Ten blok je na www.no-nekupto.sk

Dakujem.

<?php // $Id: views-view-fields.tpl.php,v 1­.6 2008/09/24 22:48:21 merlinofchaos Exp $ /**
* @file views-view-fields.tpl.php
* Default simple view template to all the fields as a row.
*
* – $view: The view in use.
* – $fields: an array of $field objects. Each one contains:
*  – $field->content: The output of the field.
*  – $field->raw: The raw data for the field, if it exists. This is NOT output safe.
*  – $field->class: The safe class id to use.
*  – $field->handler: The Views field handler object controlling this field. Do not use
* var_export to dump this object, as it can't handle the recursion.
*  – $field->inline: Whether or not the field should be inline.
*  – $field->inline_html: either div or span based on the above flag.
*  – $field->separator: an optional separator that may appear before a field.
* – $row: The raw result object from the query, with all data it fetched.
*
* @ingroup views_templates
*/ ?>

Drupal ma pre informaciu o existujucich tpl suboroch (a pre vacsi vykon) tzv theme registry. Pokial stranka bezi v beznom rezime a pridas novy tpl subor do temy tak drupal o nom nevie pokial nevynutis obnovu theme registry.
