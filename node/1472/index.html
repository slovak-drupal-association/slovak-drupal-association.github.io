Slideshow obrazkov s linkom

4 október, 2012 - 13:09 poslal Padrezz

Potreboval by som vyriešiť problém, do stránky potrebujem dosať obrázky v nejakom slide, pričom sa mi horná url bude meniť podľa obrázka a priamo odkazovať naň v slide. Skúšal som to cez views a pager, lenže tam po pridaní ďalšieho obrázku sa linky posunú a už odkazujú na iný odkaz ako pôvodne. Klasicky view zasa odkazuje iba na začiatok slide. Predom ďakujem za pomoc.

Obycajne su obrazkove galerie robene v ramci jednej node cim mas vsetky obrazky zobrazene v jednom obsahu (stranke – url cez lightbox ci fancybox). Pokial chces podobne riesenie tak musis vytvorit node type kt bude mat kazdy obrazok zvlast. Nasledne vylistovat cez views vsetky node type = img_gallery a to cez Block zobrazit pri tomto node type cez url nastavenie bloku. (Vo views je nastavenie pri obrazkoch „link to content“) Okrem toho treba nastavit URL alias setting aby si pri tom node type mal stale napr img_gallery/[title] cim zabespecis block url nastavenie.

Pokial chces mat to riesene cez 1 noda, mnoho obrazkov, rozne URL, treba to robit cez panels > node template > send argument URL a tento URL pouzit ako context filter vo View pre Image delta. Podrobne to tu nedokazem vypisat lebo je toho vela.
