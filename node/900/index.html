GMap linka na mapu pre Slovensko?

10 február, 2011 - 10:14 poslal Darinka01

Snazim sa integrovat do webu moduly GMap a Location, ktore celkom dobre kooperuju

...az na to, ze ked sa snazim, aby mi z adresy vygenerovalo linku na Google mapu, pod strankou admin/settings/location/maplinking zistujem, ze neexistuje podpora pre Slovensko

Country = "Slovakia" ... Options = "None supported."

Namiesto "None supported" by som tam rada videla "Google Maps". Nema s tym niekto skusenosti?

Dakujem krasne za odpovede, podarilo sa mi to vyriesit, takze to tu davam pre pripad potreby buducich generacii -

V module Location v adresari /supported treba vyhladat subor pre Slovensko - location.sk a doplnit do neho par funkcii, ktore tam chybaju.

Pripojila by som dany subor ako prilohu, ale sa to tu asi neda, takze to tu pastnem cele jak je:


<?php
// $Id: location.sk.inc,v 1.6 2008/01/24 09:00:03 bdragon Exp $

// Slovakia

function location_province_list_sk() {
return array('BA' => "Banskobystricky",
'BR' => "Bratislavsky",
'KO' => "Kosicky",
'NI' => "Nitriansky",
'PR' => "Presovsky",
'TC' => "Trenciansky",
'TV' => "Trnavsky",
'ZI' => "Zilinsky");
}

function location_map_link_sk_providers() {
return array(
'google' => array(
'name' => 'Google Maps',
'url' => 'http://maps.google.sk',
'tos' => 'http://www.google.sk/help/terms_maps.html',
),
);
}

function location_map_link_sk_default_providers() {
return array('google');
}

function location_map_link_sk_google($location = array()) {
$query_params = array();

foreach (array('street', 'city', 'postal_code', 'country') as $field) {
if (isset($location[$field])) {
$query_params[] = $location[$field];
}
}

if (count($query_params)) {
return ('http://maps.google.sk?q='. urlencode(implode(', ', $query_params)));
}
else {
return NULL;
}
}
