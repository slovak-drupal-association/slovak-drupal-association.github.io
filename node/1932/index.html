Ako pridať vlastné CLASS a ID pre UL a LI pre menu v téme drupalu

14 marec, 2014 - 11:36 poslal salsastevie

Ahoj DRUPAL borci…

Robím responzívnu tému do drupalu 7 a narazil som na špecifický problém pri vytváraní menu.

Mám spravenú HTML šablónu, kde používam pre zmenu pri rôznych zobrazeniach JQUERY menu daisynav.

V HTML šablóne mi menu funguje a aj stránku zobrazuje správne, ak to však chcem implementovať do drupal témy, tak narazím na problém, že drupalom generované hlavné menu má v kontajneroch UL a LI inak štrukturované pareametre CLASS a ID.

Vygooglil som, že drupalway na vyriešenie tohoto problému môže byť vytvorenie nejakej preprocess funkcie v template.php, ktorá by tam doplnila potrebné CLASS a ID.

Problém je, že takúto funkciu neviem vytvoriť. Viete mi s tým niekto pomôcť?

Momentálnu štruktúru menu v drupale mám takúto:

<div class="menu-toggle-button" data-menu-id="demo-menu">≡</div>
<ul class="menu">
    <li class="first leaf active-trail"><a href="#" class="active-trail active">Menu 1</a></li>
    <li class="leaf"><a href="#">Menu 2</a></li>
    <li class="expanded"><a href="#">Menu 3</a>
        <ul class="menu">
            <li class="first leaf"><a href="#" title="">Submenu 3.1</a></li>
            <li class="leaf"><a href="#" title="">Submenu 3.2</a></li>
            <li class="last leaf"><a href="#" title="">Submenu 3.3</a></li>
        </ul>
    </li>
    <li class="leaf"><a href="#">Menu 4</a></li>
    <li class="last expanded"><a href="#">Menu 5</a>
        <ul class="menu">
            <li class="first leaf"><a href="#" title="">Submenu 5.1</a></li>
            <li class="last leaf"><a href="#" title="">Submenu 5.2</a></li>
        </ul>
    </li>
</ul>

a potrebujem sa dopracovať k niečomu takémuto:

<div class="menu-toggle-button" data-menu-id="demo-menu">≡</div>
<ul class="menu" id="demo-menu">
    <li class="first leaf active-trail"><a href="#" class="active-trail active">Menu 1</a></li>
    <li class="leaf"><a href="#">Menu 2</a></li>
    <li class="expanded has-submenu"><a href="#">Menu 3</a>
        <ul class="menu sub-menu">
            <li class="first leaf"><a href="#" title="">Submenu 3.1</a></li>
            <li class="leaf"><a href="#" title="">Submenu 3.2</a></li>
            <li class="last leaf"><a href="#" title="">Submenu 3.3</a></li>
        </ul>
    </li>
    <li class="leaf"><a href="#">Menu 4</a></li>
    <li class="last expanded has-submenu"><a href="#">Menu 5</a>
        <ul class="menu sub-menu">
            <li class="first leaf"><a href="#" title="">Submenu 5.1</a></li>
            <li class="last leaf"><a href="#" title="">Submenu 5.2</a></li>
        </ul>
    </li>
</ul>

Čiže doplniť IDčko v hlavnom UL a v sub UL doplniť inú CLASSu.

Nejaké nápady ako to spraviť?

Vopred ďakujem za akúkoľvek pomoc.

Ďakujem za radu.

Skúsil som do svojho template.php na koniec pridať toto:

function theme_menu_tree($variables) {
  return '<ul class="menu" id="moje-menu">' . $variables['tree'] . '</ul>';
}

Ale žiadna zmena sa neprejavila stále mi to generuje

<ul class="menu">

Už to mu prestávam úplne rozumieť.

Takto vyzerá moje template.php:

<?php

/**
 * Implements hook_html_head_alter().
 * This will overwrite the default meta character type tag with HTML5 version.
 */
function MOJATEMA_html_head_alter(&$head_elements) {
  $head_elements['system_meta_content_type']['#attributes'] = array(
    'charset' => 'utf-8'
  );
}

/**
 * Insert themed breadcrumb page navigation at top of the node content.
 */
function MOJATEMA_breadcrumb($variables) {
  $breadcrumb = $variables['breadcrumb'];
  if (!empty($breadcrumb)) {
    // Use CSS to hide titile .element-invisible.
    $output = '<h2 class="element-invisible">' . t('You are here') . '</h2>';
    // comment below line to hide current page to breadcrumb
        $breadcrumb[] = drupal_get_title();
    $output .= '<nav class="breadcrumb">' . implode(' » ', $breadcrumb) . '</nav>';
    return $output;
  }
}

/**
 * Override or insert variables into the page template.
 */
function MOJATEMA_preprocess_page(&$vars) {
  if (isset($vars['main_menu'])) {
    $vars['main_menu'] = theme('links__system_main_menu', array(
      'links' => $vars['main_menu'],
      'attributes' => array(
        'class' => array('links', 'main-menu'),
      ),
      'heading' => array(
        'text' => t('Main menu'),
        'level' => 'h2',
        'class' => array('element-invisible'),
      )
    ));
  }
  else {
    $vars['main_menu'] = FALSE;
  }
  if (isset($vars['secondary_menu'])) {
    $vars['secondary_menu'] = theme('links__system_secondary_menu', array(
      'links' => $vars['secondary_menu'],
      'attributes' => array(
        'class' => array('links', 'secondary-menu', 'clearfix'),
      ),
      'heading' => array(
        'text' => t('Secondary menu'),
        'level' => 'h2',
        'class' => array('element-invisible'),
      )
    ));
  }
  else {
    $vars['secondary_menu'] = FALSE;
  }
}
/**
 * Duplicate of theme_menu_local_tasks() but adds clearfix to tabs.
 */
function MOJATEMA_menu_local_tasks(&$variables) {
  $output = '';

  if (!empty($variables['primary'])) {
    $variables['primary']['#prefix'] = '<h2 class="element-invisible">' . t('Primary tabs') . '</h2>';
    $variables['primary']['#prefix'] .= '<ul class="tabs primary clearfix">';
    $variables['primary']['#suffix'] = '</ul>';
    $output .= drupal_render($variables['primary']);
  }
  if (!empty($variables['secondary'])) {
    $variables['secondary']['#prefix'] = '<h2 class="element-invisible">' . t('Secondary tabs') . '</h2>';
    $variables['secondary']['#prefix'] .= '<ul class="tabs secondary clearfix">';
    $variables['secondary']['#suffix'] = '</ul>';
    $output .= drupal_render($variables['secondary']);
  }
  return $output;
}

/**
 * Override or insert variables into the node template.
 */
function MOJATEMA_preprocess_node(&$variables) {
  $node = $variables['node'];
  if ($variables['view_mode'] == 'full' && node_is_page($variables['node'])) {
    $variables['classes_array'][] = 'node-full';
  }
}

function MOJATEMA_page_alter($page) {
  // <meta name="viewport" content="width=device-width, initial-scale=1, maximum-scale=1"/>
  $viewport = array(
    '#type' => 'html_tag',
    '#tag' => 'meta',
    '#attributes' => array(
    'name' =>  'viewport',
    'content' =>  'width=device-width'
    )
  );
  drupal_add_html_head($viewport, 'viewport');
}
function MOJATEMA_menu_tree(&$variables) {
return '<ul class="menu-list" id="demo-menu">' . $variables['tree'] . '</ul>';
}

                      $main_menu_tree = menu_tree(variable_get('menu_main_links_source', 'main-menu'));
          }
          print drupal_render($main_menu_tree);
          ?>
